# JobSonar HR Monitoring System

Система автоматизированного мониторинга вакансий в сфере управления персоналом (HR) на платформе HeadHunter.

## Основной функционал

* **Двухуровневая фильтрация**: Вакансия проходит отбор только при одновременном наличии в названии HR-контекста (объект) и управленческого статуса (роль).
* **Интеллектуальный поиск**: Алгоритм учитывает корни слов, что позволяет находить вакансии в разных падежах и формах без потери точности.
* **Гибридный режим работы**:
    * Мониторинг целевого списка (Whitelist) из 400+ крупнейших компаний за последние 14 дней.
    * Автоматическая глобальная «докачка» по всему рынку при малом количестве результатов (только удаленная работа с высоким порогом оплаты за последние 3 дня).
* **Очистка данных**: Жесткий список стоп-слов исключает смежные отрасли (IT, продажи, финансы, административно-хозяйственная деятельность) и линейные позиции.

## Технические особенности

* **База данных**: SQLite используется для хранения истории отправленных вакансий и предотвращения дублирования.
* **Форматирование**: Компактный вид сообщений в Telegram с гиперссылками на вакансии и автоматическим определением формата работы (офис, гибрид, удаленно).
* **Пагинация**: Полный обход всех доступных страниц выдачи API HeadHunter.

## Настройка и запуск

1. Установите зависимости: `pip install requests python-dotenv`
2. Настройте файл `.env` (TG_TOKEN, TG_CHAT_ID).
3. Актуализируйте список компаний в `whitelist.py`.
4. Запустите основной скрипт: `python main.py`
